# Implementation Plan: Scryfall Card Art Integration

**Branch**: `016-scryfall-card-art` | **Date**: 2026-03-24 | **Spec**: [spec.md](spec.md)
**Input**: Feature specification from `/specs/016-scryfall-card-art/spec.md`

## Summary

Display Scryfall card art images in the React game UI by constructing CDN image URLs from the existing `scryfall_id` field on the `Card` model, and make the full game board usable on mobile phones. No backend changes are required. The `CardView` component gains an art image with text overlays for gameplay stats; tokens and cards without `scryfall_id` fall back to text display. The board layout is made responsive by removing the fixed 300px sidebar on mobile and scaling card dimensions for small viewports.

**Constraints**: No new backend endpoints; no new npm dependencies; must not break existing text fallback
**Scale/Scope**: ~40–100 visible cards per game session; 2–6 players in Commander

## Implementation Notes

### `frontend/src/utils/scryfall.ts` (new)

```typescript
export type ScryfallImageSize = 'small' | 'normal' | 'large'
export type ScryfallFace = 'front' | 'back'

export const CARD_BACK_URL =
  'https://cards.scryfall.io/small/back/0/0/0aeebaf5-8c7d-4636-9e82-8c27447861f7.jpg'

export function scryfallImageUrl(
  scryfallId: string,
  face: ScryfallFace = 'front',
  size: ScryfallImageSize = 'small'
): string {
  const a = scryfallId[0]
  const b = scryfallId[1]
  return `https://cards.scryfall.io/${size}/${face}/${a}/${b}/${scryfallId}.jpg`
}
```

### `CardView.tsx` logic

```typescript
// Determine image URL
let imageUrl: string | null = null
if (permanent?.is_face_down) {
  imageUrl = CARD_BACK_URL
} else if (card.scryfall_id) {
  imageUrl = scryfallImageUrl(card.scryfall_id)
}
```

- Apply `imageUrl` as inline `style.backgroundImage` on the `.card` div
- Add `card-has-art` class when `imageUrl` is set, for CSS targeting
- Use `onError` on a hidden `<img>` to detect load failure and clear `imageUrl`

### `card.css` additions

```css
/* When art is available: cover fill, darken for text legibility */
.card.card-has-art {

}
```
